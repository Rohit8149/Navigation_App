Features

Everything You Need for Hands-Free Navigation

A complete AI-powered accessibility stack built into one lightweight app.

AI-Powered Navigation

Sends your command to Groq LLaMA which returns a step-by-step path through Android Settings. Re-routes automatically if a step is missing.

Groq LLaMA 3

Smart Node Matching

Word-boundary-aware fuzzy matching with Levenshtein scoring. Never confuses "Font size" with "Display size" — strict thresholds prevent all false positives.

Multi-Strategy Fallback

Floating Overlay UI

A persistent floating icon lives on top of all apps. Tap to type a command. Status bar shows real-time step progress. Help button triggers AI re-routing.

Glass Morphism Dark UI

Developer Logging

Real-time in-app log viewer captures navigation decisions, match scores, API responses and search outcomes — no ADB required for debugging.

DevLogManager

Navigation Guard

Detects when you leave Settings and pauses navigation. Distinguishes real app exits from transient system UI overlays like volume popups or notifications.

No False Exits

Agentic Executor

Maintains full exploration history to prevent loops. Enforces root-to-leaf order. Auto-scrolls to find off-screen elements before declaring a step "not found".

Loop-Free Navigation

Live UI Tree Viewer

Built-in screen that dumps and displays the live accessibility UI tree of any foreground app — see exactly what the AI and navigator are working with.

On-Device Debugger

Accessibility-First

Built on Android's AccessibilityService API. Works system-wide without root. Captures and interacts with any app's UI tree in real time.

No Root Required
How It Works

From Command to Action in Seconds

Type a natural language command — the AI figures out the path and the app does the rest.

01

You Type a Command

Open the floating dialog and type something like "Open Wi-Fi settings" or "Change font size".

02

AI Plans the Path

Groq LLaMA receives your command + the live UI tree and returns a step-by-step navigation plan through the Settings hierarchy.

03

Smart Node Matching

The NodeMatcher finds the exact UI element to tap using fuzzy word-boundary matching — no false positives, handles typos gracefully.

04

Tap & Navigate

The accessibility service performs the actual taps on screen, step by step, showing you live progress in the status overlay.

How to Install

  1. 1 Download the APK file
  2. 2 Enable "Install Unknown Apps" in Settings
  3. 3 Open the downloaded APK and tap Install
  4. 4 Go to Settings → Accessibility → NavigationApp → Enable
  5. 5 Open the app, tap Start Assistant and you're ready!
